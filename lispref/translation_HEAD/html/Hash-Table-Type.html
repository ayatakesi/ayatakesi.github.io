2.4.12 ハッシュテーブル型 ¶

ハッシュテーブルは非常に高速な照合テーブルの一種で、キーを対応する値にマップするalistと似ていますがより高速です。ハッシュテーブルのプリント表現では、以下のようにハッシュテーブルのプロパティと内容を指定します: 

(make-hash-table)
     ⇒ #s(hash-table)

ハッシュテーブルについての詳細はハッシュテーブルを参照してください。
