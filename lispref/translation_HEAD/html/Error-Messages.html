1.3.5 エラーメッセージ ¶

エラーをシグナルする例もあります。これは通常、エコーエリアにエラーメッセージを表示します。エラーメッセージの行は‘error→’で始まります。‘error→’自体は、エコーエリアに表示されないことに注意してください。 

(+ 23 'x)
error→ Wrong type argument: number-or-marker-p, x
