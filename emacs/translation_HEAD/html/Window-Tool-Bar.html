22.9 ウィンドウツールバー ¶

コマンドglobal-window-tool-bar-modeはそれぞれのウィンドウ上端のツールバーの表示を切り替えます。有効なら複数のウィンドウで独自のツールバーを同時に表示できます。スペース節約のために、ボタンがないウィンドウツールバー(tool-bar-mapがnil)は非表示になります。

一部のバッファーにたいしてのみウィンドウツールバーの表示を切り替えたい場合には、それらのバッファーでコマンドwindow-tool-bar-modeを実行してください。これはモードフックにセットしておくと便利です。たとえばファイルと無関係なバッファーでのみカスタマイズしたウィンドウツールバーを表示したければ、initファイル(Emacs初期化ファイルを参照)に以下を追加します: 

(add-hook 'special-mode-hook 'window-tool-bar-mode)

Emacsではフレーム上端に単一のツールバーも表示できます(ツールバーを参照)。

ウィンドウツールバーはタブラインと同じスペースに表示されるので、Lispでtab-line-formatに(:eval (window-tool-bar-string))を追加するようtab-line-formatの値をカスタマイズしていないかぎり、表示されるのは常にどちらか一方だけだということに注意してください。Mode Line Format in The Emacs Lisp Reference Manualを参照してください。
