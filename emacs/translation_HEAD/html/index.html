The Emacs Editor

Emacsは先進的、拡張可能、カスタマイズ可能な、セルフドキュメント方式のエディターです。このinfoファイルでは、Emacsでの編集方法や Emacsのカスタマイズ方法について説明します。GNU Emacs 29.1に対応します。

Emacsの拡張に関しては、Emacs Lisp in The Emacs Lisp Reference Manualを参照してください。
