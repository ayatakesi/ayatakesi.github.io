Move to the previous hunk-start (@code{diff-hunk-prev}). With prefix argument @var{n}, move back to the @var{n}th previous hunk. Like @kbd{M-n}, this has the side-effect of refining command refines the hunk you move to, unless to if you disable Diff Auto-Refine mode. set @code{diff-refine} to the symbol @code{navigation}.
