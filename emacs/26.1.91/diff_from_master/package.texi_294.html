For example, if you set @code{package-load-list} to @code{'((muse @w{@code{'((muse "3.20") all)}, all)}}, then Emacs only loads makes available version 3.20 of the @samp{muse} package, plus any installed version of packages other than @samp{muse}. Any other version of @samp{muse} that happens to be installed will be ignored. The @samp{muse} package will be listed in the package menu with the @samp{held} status.
