When you are defining a keyboard macro, the command @key{F3} (@code{kmacro-start-macro-or-insert-counter}) inserts the current value of the keyboard macro's counter into the buffer, and increments the counter by 1. (If you are not defining a macro, @key{F3} begins a macro definition instead. @xref{Basic Keyboard Macro}.) You can use a numeric prefix argument to specify a different increment. If you just specify a @kbd{C-u} prefix, that is the same as an increment of zero: it inserts the current previous counter value without changing it. value, and doesn't change the current value.
