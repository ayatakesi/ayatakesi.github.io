Once a package is downloaded and installed, it is @dfn{loaded} into made available to the current Emacs session. Loading a package is not quite the same as loading a Lisp library (@pxref{Lisp Libraries}); loading Making a package available adds its directory to @code{load-path} and loads its autoloads. The effect of a package's autoloads varies from package to package. Most packages just make some new commands available, while others have more wide-ranging effects on the Emacs session. For such information, consult the package's help buffer.
