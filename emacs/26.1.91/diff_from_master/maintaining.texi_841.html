To compare two arbitrary revisions of the current VC fileset, whole trees, call @code{vc-diff} @code{vc-root-diff} with a prefix argument: @kbd{C-u C-x v =}. D}. This prompts for two revision IDs (@pxref{VCS Concepts}), and displays a diff between those versions of the fileset. This will not work reliably for multi-file VC filesets, if the version control system is file-based rather than changeset-based (e.g., CVS), since then revision IDs for different files would entire version-controlled directory trees (RCS, SCCS, CVS, and SRC do not be related in any meaningful way. support this feature).
